# 💊 PharmaCare  
### Full Stack Pharmacy Management & Inventory Automation System

PharmaCare is a full-stack pharmacy management system developed using Java, JSP, Servlets, and MySQL.  
It helps pharmacy shopkeepers manage inventory, automate stock tracking, and streamline distributor communication.

---

## 🚀 Key Features

- 🔐 Secure Login & Authentication System
- 💊 Medicine Inventory Management
- 📦 Automatic Stock Update After Sales
- ⚠️ Low-Stock Email Alerts (JavaMail API Integration)
- 🧾 Bill Generation & Receipt Management
- 📧 Automated Distributor Order Email System
- 📊 Real-Time Inventory Visibility

## 🏗️ Project Architecture

This project follows MVC Architecture:

- Model → Java Beans & DAO
- View → JSP Pages
- Controller → Servlets

## ⚙️ How To Run The Project

1. Clone the repository
2. Import into Eclipse as Dynamic Web Project
3. Configure Apache Tomcat Server
4. Create MySQL Database
5. Import `database.sql`
6. Update DB credentials in connection file
7. Run on server
