# 💊 PharmaCare - Pharmacy Management & Inventory Automation System

🚀 Full Stack Java Project | Inventory Automation | Email Alerts | Billing System  

---

## 📌 Overview
PharmaCare is a full-stack pharmacy management system designed to automate inventory tracking, billing, and distributor communication. It replaces manual processes with a digital system, improving accuracy, efficiency, and real-time stock visibility.

---

## ❗ Problem Statement
Pharmacy shops often face:
- Manual inventory tracking  
- Stock mismanagement  
- Delayed communication with distributors  
- Human errors in billing and stock updates  

---

## 💡 Solution
PharmaCare provides an automated system to manage medicines, track inventory, generate bills, and send real-time email notifications for stock updates and distributor orders.

---

## ✨ Key Features

### 🔐 Authentication
- Secure login system for authorized access  

### 💊 Inventory Management
- Add, update, delete medicines  
- Real-time stock tracking  
- Automatic stock updates after sales  

### 📦 Stock & Alerts
- Low-stock email alerts using JavaMail API  
- Automated distributor order email system  

### 🧾 Billing System
- Generate bills and receipts  
- Maintain transaction records  

---

## 📊 Impact
- 📈 Improved inventory visibility by **80%**  
- ❌ Reduced manual tracking errors by **70%**  
- ⚡ Automated distributor communication  

---

## 🛠 Tech Stack
- **Backend:** Java, JSP, Servlets, JDBC  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** MySQL  
- **Tools:** Git, Apache Tomcat  
- **API:** JavaMail API  

---

## 📂 Project Architecture
- Model → Java Beans  
- View → JSP Pages  
- Controller → Servlets  
- DAO → Database operations using JDBC
